Utilizando BreakOut Detection de Twitter para detección de anomalías en series de datos

Valery Calderón Briz

  • 23 September 2017, 1 p.m. - 1:30 p.m.
  • Room Skyscanner
  • Idioma: es

La librería "BreakOut Detection" está desarrollada en R por lo que aprendemos a llamarla desde python y a utilizar correctamente los parámetros que condicionan el algoritmo y nos ayudan a tener una correcta detección de anomalías dependiendo del caso que se presente.

El BreakOut Detection es muy útil para obtener una detección de anomalías en una serie de datos respecto al tiempo, está fue desarrollada por el equipo de Twitter y funciona correctamente sin necesidad de una gran complejidad o muchos recursos.
