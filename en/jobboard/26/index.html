Python / Django Developer

Desde Kernel estamos buscando un compañero desde recién graduado hasta experimentado programador para unirse a nuestro equipo técnico como Python / Django Developer. El candidato participará en el desarrollo de varios proyectos de análisis de datos.

El candidato deseado debe ser: apasionado por la tecnología, con iniciativa, proactivo, buen trabajador en equipo, dispuesto a continuar aprendiendo permanentemente e innovando cada dia, alguien que entienda la importancia de escribir un código claro, descriptivo y reaprovechable.

Kernel Analytics ofece la posibilidad de trabajar en un ambiente innovador, joven y dinámico (tanto en cuanto a las personas en la compañía como en los lenguajes de programación, herramientas y sistemas utilizados en los proyectos).

Requisitos

  • Grado en Informática
  • Sólidos conocimientos de programación, idealmente en Python y Javascript. Programación orientada a objetos
  • HTML, CSS
  • Experiencia trabajando con RESTful APIs
  • Experiencia en desarrollo web (back-end y front-end), preferiblemente utilizando el framework Django/Python
  • Conocimiento de SQL y manejo de BBDD relacionales
  • Experiencia testeando en Django y Javascript con frameworks como Jasmine, Mocha...
  • Experiencia trabajando con Git

Qué ofrecemos

  • Contrato indefinido
  • Desarrollo profesional
  • Salario competitivo
  • Beneficios sociales
  • Progresión de carrera
  • Plan de formación
  • Fruta fresca y café gratuitos

¿Quieres saber más? Acércate a nuestro stand durante la conferencia, estaremos todo el fin de semana allí para resolver todas tus dudas o charlar un rato. Además, organizamos la Hackathon que se celebra dentro de PyConEs, y gran parte de nuestro equipo asiste a la conferencia: les puedes reconocer por las camisetas oficiales de la #PythonHack2017. No dudes en acercarte a uno de nosotros, y saber como es trabajar en Kernel de primera mano ;)
