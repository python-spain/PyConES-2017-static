Come learn about the Python Software Foundation!

Naomi Ceder

  • 23 setembro 2017, 15:00 - 15:30
  • Sala Intelygenz
  • Idioma: en

The PSF plays a key role in fostering the Python community around the world. Come find out what the PSF does (and doesn't do), how it works, how the PSF can support your community, and how you can be part of the PSF. We'll describe the membership model and how everyone can be a member, and many people can be voting members. We will also talk about the PSF Grants program, and answer any questions you might have. Everyone is welcome, both non-members and members! PSF stickers will also be available!
